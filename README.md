# Task Manager
- A simple and responsive Task Manager web application developed using Spring Boot, MySQL and Bootstrap.
- The application helps users to manage daily tasks by allowing them to create, update, delete, search, and filter tasks based on status through a interface.

# Features
- Create new tasks
- Edit existing tasks
- Delete tasks
- Search tasks by title or description
- Filter tasks by status

# Technologies Used
- Java Spring Boot
- MySQL
- Bootstrap

# MVC Architecture
This application uses MVC architecture.



# Run the Project

- Clone Repository
git clone https://github.com/ayishaselin/Task-Management-System.git

- Run Application
./mvnw spring-boot:run

- Open in Browser
http://localhost:8080/tasks
